Frequently asked questions

Here are some questions that we gat asked very often, and the answers to them

1.How does the elo system work?

  Elo is a score given to a weaponball. Every weaponball starts with an Elo of 1000. After every battle, a set amount of Elo is calculated to be added to
the winner's Elo and subtracted from the loser's. The things the script takes into consideration while calculating the Elo subtracted/added:

    a) The Elo difference: if a weaponball wins against an opponent that has a higher Elo than him, he gains more Elo than he would've gained from a win
    against a weaponball with the same/lower elo. A similar concept applies to losses, if a weaponball loses against a opponent with lower rating opponent
    it will lose more Elo than if he lost to an opponent with the same/higher Elo rating;

    b) The Elo cap: the amount of Elo you lose/gain is capped at 64 in order to make the rankings more stable and less chaotic.




2.What battles are taken in consideration?

  The only batles taen in consideration are the regular 1v1s, extended 1v1s, and best of 3 1v1s, also I want to mention that for the best of 3 1v1s we
count only the final result.

json that contains all of them, run the python code, and it should automaticaly update.
